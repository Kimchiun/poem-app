# 감성 시 공유 플랫폼 (Poem App)

익명으로 누구든지 감성적인 시와 사진을 공유할 수 있는 웹 애플리케이션입니다.
Supabase를 백엔드로 사용하여 실시간으로 데이터를 저장하고 공유합니다.

## ✨ 주요 기능

*   **시 작성 (Create)**: 제목, 내용, 그리고 분위기에 맞는 사진을 업로드하여 시를 작성할 수 있습니다.
*   **비밀번호 보호 (Security)**: 게시물 작성 시 비밀번호를 설정하여, 본인만 삭제할 수 있도록 보호합니다.
*   **감성적인 UI**: Glassmorphism 디자인과 부드러운 애니메이션을 적용하여 몰입감을 높였습니다.
*   **삭제 기능 (Delete)**: 커스텀 모달창을 통해 비밀번호를 확인하고 게시물을 안전하게 삭제합니다.
*   **반응형 디자인**: 데스크탑과 모바일 모두에서 최적화된 화면을 제공합니다.

## 🛠️ 기술 스택

*   **Frontend**: React, Vite
*   **Styling**: Vanilla CSS (Glassmorphism, Animations)
*   **Backend & Database**: Supabase (PostgreSQL)
*   **Deployment**: Vercel

## 🚀 설치 및 실행 방법

1.  **프로젝트 클론**
    ```bash
    git clone https://github.com/Kimchiun/poem-app.git
    cd poem-app
    ```

2.  **의존성 설치**
    ```bash
    npm install
    ```

3.  **환경 변수 설정**
    프로젝트 루트에 `.env` 파일을 생성하고 Supabase 키를 입력하세요.
    (`.env.example` 파일 참고)
    ```env
    VITE_SUPABASE_URL=your_supabase_url
    VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
    ```

4.  **개발 서버 실행**
    ```bash
    npm run dev
    ```

## 🗄️ 데이터베이스 설정 (Supabase)

Supabase SQL Editor에서 다음 쿼리를 실행하여 테이블을 생성해야 합니다.

```sql
CREATE TABLE poems (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT TIMEZONE('utc', NOW()),
  title TEXT NOT NULL,
  text TEXT NOT NULL,
  author TEXT DEFAULT 'Anonymous',
  password TEXT,
  image TEXT,
  likes INT DEFAULT 0,
  comments INT DEFAULT 0
);

-- RLS 비활성화 (프로토타입용)
ALTER TABLE poems DISABLE ROW LEVEL SECURITY;
```
